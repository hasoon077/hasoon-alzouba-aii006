Web Design Analysis

Two websites examined — one appealing, one unappealing — evaluated across five criteria.

Website One

01
Appealing

Apple

https://www.apple.com

a. URL https://www.apple.com
b. Does it use tables? Apple.com uses tables only for structured data such as product specification comparisons. Tables are not used for page layout — all layout is CSS-based.
c. Does it use CSS? Yes, extensively. CSS handles the full page layout via Flexbox and Grid, typography scaling, color theming, hover transitions, scroll animations, and responsive breakpoints for all device sizes.
d. Appealing — three reasons
  • Minimalist layout with generous whitespace. Every element feels deliberate. Nothing competes for attention unnecessarily, making the page instantly readable and visually calming.
  • Consistent, refined typography. Apple's custom typefaces are used with precise sizing hierarchy — bold headings, readable body text, and consistent rhythm that reinforces the premium brand.
  • Smooth, purposeful animations. Scroll-triggered effects and product showcases feel cinematic rather than gimmicky. They guide the eye naturally without slowing navigation.
e. Improvements Not applicable — this website is appealing. The only minor critique is that deeply nested product categories can require several taps to reach on mobile.

Website Two

02
Unappealing

Ling's Cars

https://www.lingscars.com

a. URL https://www.lingscars.com
b. Does it use tables? Yes — both for page layout (an outdated practice) and to organize car listings. The combination makes the page structure visually confusing and hard to parse.
c. Does it use CSS? Yes, but ineffectively. CSS is applied for colors and font sizes, but the choices are chaotic — clashing neon backgrounds, multiple competing text colors per section, and no consistent spacing or visual hierarchy.
d. Unappealing — three reasons
  • Extreme visual chaos. The homepage bombards visitors with flashing GIF animations, autoplaying videos, and dozens of competing banners. Every element shouts at the same volume — there is no focal point.
  • No coherent color scheme or typography. Text appears in red, green, yellow, white, and blue within the same section. Font sizes are inconsistent and many headings are in ALL CAPS with decorative fonts that hurt readability.
  • Confusing navigation and unclear purpose. A first-time visitor cannot quickly identify how to browse cars or get a quote. Core user goals require significant effort due to information buried under promotional noise.
e. Improvements
  • Establish a clean 2–3 color palette. Remove clashing neon backgrounds and replace with neutral whites or light grays that let car photography stand out.
  • Redesign navigation. A clear top menu with logical categories and a visible "Get a Quote" CTA would immediately improve usability without scrolling.
  • Remove animated GIFs and autoplaying media. These distract from the product and slow page load. Static images with a single intentional animation would create a more professional impression.
